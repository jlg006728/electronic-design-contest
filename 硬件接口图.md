## 二、MSPM0G3507 LaunchPad 引脚分配总表

### 电机驱动 (TB6612FNG)

| MSPM0 引脚 | 外设功能 | TB6612 引脚 | 说明 |
|------------|----------|-------------|------|
| **PB0** | GPIO (输出) | AIN1 | 左电机方向1 |
| **PB1** | GPIO (输出) | AIN2 | 左电机方向2 |
| **PB2** | GPIO (输出) | BIN1 | 右电机方向1 |
| **PB3** | GPIO (输出) | BIN2 | 右电机方向2 |
| **PB4** | GPIO (输出) | STBY | 使能 (HIGH=工作) |
| **PA12** | TIMG0 CCP0 | PWMA | 左电机 PWM (20kHz) |
| **PA13** | TIMG0 CCP1 | PWMB | 右电机 PWM (20kHz) |

> 注意: PA12/PA13 是 TIMG0 的**唯一**可用输出引脚，不可更改。

### 巡线传感器 (XD-01 TCRT5000 ×7)

使用 **DO (数字输出)** 接 GPIO，避免 ADC 引脚不足问题。

| MSPM0 引脚 | 传感器 | 说明 |
|------------|--------|------|
| **PA2** | S1 (最左) | GPIO 输入，内部上拉 |
| **PA3** | S2 | GPIO 输入 |
| **PA4** | S3 | GPIO 输入 |
| **PA5** | S4 (中间) | GPIO 输入 |
| **PA6** | S5 | GPIO 输入 |
| **PA7** | S6 | GPIO 输入 |
| **PA8** | S7 (最右) | GPIO 输入 |

> XD-01 模块自带 LM393 比较器 + 电位器调阈值，DO 输出 TTL 电平。
> 黑线=HIGH，白底=LOW。7 路传感器可做加权平均计算偏差。

### 舵机云台 (MG90S ×2)

| MSPM0 引脚 | 外设功能 | 舵机 | 说明 |
|------------|----------|------|------|
| **PA3** | TIMG8 CCP0 | 云台水平(X) | 50Hz PWM, 500-2500μs |
| **PA4** | TIMG8 CCP1 | 云台俯仰(Y) | 50Hz PWM, 500-2500μs |

> ⚠️ 原配置使用不存在的 TIMG2，修正为 TIMG8。
> TIMG8 是 MSPM0G3507 上可用的定时器，CCP0/CCP1 在 PA3/PA4。

**⚠️ 引脚冲突警告**: PA3/PA4 同时被分配给巡线传感器 S2/S3 和舵机 PWM。
需要在传感器和舵机之间做取舍。**建议方案：**

**方案A (推荐)**: 巡线传感器改为纯数字读取，只用 PA2, PA5-PA8, PB9, PB10 共7个GPIO，PA3/PA4 留给舵机。

**方案B**: 使用 TIMG12 (PB13/PA31) 驱动舵机，PA3/PA4 留给传感器。

> 以下按**方案A**继续分配。

### 修正后的巡线传感器分配 (方案A)

| MSPM0 引脚 | 传感器 | 说明 |
|------------|--------|------|
| **PA2** | S1 (最左) | GPIO 输入 |
| **PA5** | S2 | GPIO 输入 |
| **PA6** | S3 | GPIO 输入 |
| **PA7** | S4 (中间) | GPIO 输入 |
| **PA8** | S5 | GPIO 输入 |
| **PB9** | S6 | GPIO 输入 |
| **PB10** | S7 (最右) | GPIO 输入 |

### 编码器 (WHEELTEC R3X, 500线GMR, ×4倍频)

| MSPM0 引脚 | 编码器 | 说明 |
|------------|--------|------|
| **PC0** | 左轮 A相 | GPIO 中断 (双沿) |
| **PC1** | 左轮 B相 | GPIO 输入 (判断方向) |
| **PC2** | 右轮 A相 | GPIO 中断 (双沿) |
| **PC3** | 右轮 B相 | GPIO 输入 (判断方向) |

> GMR 编码器为推挽输出，无需上拉电阻。
> 500线 × 30减速比 × 4倍频 = 60000 边沿/圈

### UART 通信 (OpenMV)

| MSPM0 引脚 | 外设功能 | OpenMV 引脚 | 说明 |
|------------|----------|-------------|------|
| **PA10** | UART0 TX | P4 (RX) | MSPM0→OpenMV 请求 |
| **PA11** | UART0 RX | P5 (TX) | OpenMV→MSPM0 坐标数据 |

> 波特率 115200, 8N1。原配置 PA12 为 TX 是错误的 (PA12 已用于 TIMG0)。

### I2C (MPU6050, 可选)

| MSPM0 引脚 | 外设功能 | MPU6050 引脚 | 说明 |
|------------|----------|-------------|------|
| **PB2** | I2C1 SDA | SDA | ⚠️ 与 BIN1 冲突 |
| **PB3** | I2C1 SCL | SCL | ⚠️ 与 BIN2 冲突 |

> ⚠️ I2C0 固定在 PA0/PA1 (已被 LaunchPad 占用)。I2C1 在 PB2/PB3，与电机方向引脚冲突。
> **建议**: 暂不使用 MPU6050，用编码器里程计替代。如必须使用，需重新分配电机方向引脚到其他 GPIO。

### 按键与指示灯

| MSPM0 引脚 | 功能 | 说明 |
|------------|------|------|
| **PB8** | MODE 按键 | GPIO 中断 (下降沿)，循环选择模式 |
| **PB11** | START 按键 | GPIO 中断 (下降沿)，启动/停止 |
| **PA15** | LED 状态指示 | GPIO 输出，闪烁指示当前模式 |

### 激光模块

| MSPM0 引脚 | 功能 | 说明 |
|------------|------|------|
| **PA20** | 激光控制 | GPIO 输出，HIGH=开启 |

> 激光模块供电: 3.3V 或 5V (看模块规格)，信号线串 100Ω 限流电阻。

## 四、电源系统设计

```
2S LiPo 7.4V ──┬── 双路开关1 ──→ TB6612 VM ──→ 驱动左右电机
               │                  │
               │                  └── TB6612 板载稳压 ──→ 5V OUT
               │                                          │
               │                          ┌───────────────┤
               │                          │               │
               │                     LaunchPad 5V     MG90S ×2
               │                     (内部3.3V稳压)   (舵机供电)
               │                          │
               │                     MSPM0G3507
               │                     (3.3V)
               │
               └── 双路开关2 ──→ 独立5V稳压 ──→ 激光模块
                                (如需要)
```

### 各模块供电需求

| 模块 | 电压 | 典型电流 | 峰值电流 | 供电来源 |
|------|------|----------|----------|----------|
| MSPM0G3507 | 3.3V | 20mA | 50mA | LaunchPad 板载稳压 |
| TB6612FNG (VM) | 7.4V | — | 1.2A/通道 | 电池直连 |
| TT电机 ×2 | 6-7.4V | 200mA | 800mA | TB6612 输出 |
| MG90S ×2 | 5V | 150mA | 750mA | 5V稳压 |
| TCRT5000 ×7 | 3.3V | 70mA | 140mA | LaunchPad 3.3V |
| OpenMV H7 | 3.3V | 80mA | 120mA | LaunchPad 3.3V |
| 激光模块 | 5V | 20mA | 50mA | 5V稳压 |
| **合计** | | **~540mA** | **~3A** | |

> 电池建议: 2S 7.4V 1000mAh 以上，支持 5C+ 放电 (5A+)

## 五、TB6612FNG 接线详图

```
                    TB6612FNG 模块
                 ┌──────────────────┐
    VM (7.4V) ───┤ VM          AO1 ├──→ 左电机 +
                 │                  │
    GND ─────────┤ GND         AO2 ├──→ 左电机 -
                 │                  │
    PB0 ─────────┤ AIN1        BO1 ├──→ 右电机 +
                 │                  │
    PB1 ─────────┤ AIN2        BO2 ├──→ 右电机 -
                 │                  │
    PB2 ─────────┤ BIN1           │
                 │                  │
    PB3 ─────────┤ BIN2    5V OUT ├──→ 5V 供电 (舵机/激光)
                 │                  │
    PA12 ────────┤ PWMA           │
                 │                  │
    PA13 ────────┤ PWMB           │
                 │                  │
    PB4 ─────────┤ STBY           │
                 └──────────────────┘

真值表:
  AIN1=H, AIN2=L, PWM>0 → 左电机正转
  AIN1=L, AIN2=H, PWM>0 → 左电机反转
  AIN1=H, AIN2=H, PWM>0 → 刹车
  STBY=L → 全部停止
```

## 六、分步上电验证流程 (防烧毁)

> ⚠️ 严格按照顺序操作，每步确认后再进行下一步。

### 第0步: 目视检查 (不通电)

- [ ] 检查所有焊点有无虚焊、短路
- [ ] 用万用表测量电池电压 (应为 7.4-8.4V)
- [ ] 用万用表测量各模块 VCC-GND 之间有无短路
- [ ] 确认 TB6612 模块稳压输出为 5V (不通电池，用万用表通断档确认线路)

### 第1步: LaunchPad 单独上电

- [ ] USB 连接电脑，确认 CCS 能识别设备
- [ ] 下载一个 LED 闪烁测试程序
- [ ] 确认 3.3V 和 5V 引脚电压正常

### 第2步: 传感器模块测试 (不接电机)

- [ ] 7 个 TCRT5000 模块接 3.3V 供电
- [ ] 用万用表测量 DO 输出: 白纸上应为 LOW，黑线上应为 HIGH
- [ ] 调节每个模块的蓝色电位器，使黑白切换阈值合适
- [ ] 下载测试程序读取 7 路 GPIO 状态，串口打印

### 第3步: 舵机测试 (不接电机)

- [ ] MG90S ×2 接 5V 独立供电 (不经过 LaunchPad)
- [ ] 信号线接 PA3/PA4
- [ ] 下载测试程序: 输出 1500μs (中位)，确认舵机归中
- [ ] 测试 500-2500μs 范围，确认舵机转动平滑无抖舵

### 第4步: 电机驱动测试 (不装轮胎)

- [ ] TB6612 接电池 7.4V (VM)
- [ ] STBY 接 HIGH
- [ ] 先不接 PWM，只测试方向引脚: AIN1=H, AIN2=L → 电机应转动
- [ ] 接 PWM (PA12)，从低占空比 (10%) 开始，逐步增大
- [ ] 确认两个电机都能正反转
- [ ] 测试刹车 (AIN1=H, AIN2=H)

### 第5步: 编码器测试

- [ ] 电机转动时，用示波器或逻辑分析仪观察编码器 A/B 相波形
- [ ] 下载测试程序: GPIO 中断计数，串口打印脉冲数
- [ ] 手转一圈，确认脉冲数 ≈ 500 (单相) 或 ≈ 2000 (4倍频)

### 第6步: OpenMV 通信测试

- [ ] OpenMV 连接电脑，运行颜色追踪脚本
- [ ] UART 连接 PA10(TX)/PA11(RX)
- [ ] 下载测试程序: 发送请求，接收坐标数据，串口打印

### 第7步: 激光模块测试

- [ ] 激光模块接 5V，信号线串 100Ω 接 PA20
- [ ] PA20 输出 HIGH，确认激光亮
- [ ] 确认光斑直径 ≤0.5cm (调焦)

### 第8步: 整机联调

- [ ] 装轮胎，放在赛道上
- [ ] 测试巡线 (基本要求1)
- [ ] 测试静态瞄准 (基本要求2/3)
- [ ] 测试行进中瞄准 (发挥部分)

## 七、与原配置的差异汇总

| 项目 | 原 config.h (错误) | 修正后 | 原因 |
|------|---------------------|--------|------|
| 巡线传感器 | PA0-PA6 (ADC) | PA2,PA5-PA8,PB9,PB10 (GPIO) | PA0-PA6 不是 ADC 引脚 |
| 电机 PWM | PB6/PB7 (TIMG0) | PA12/PA13 (TIMG0) | PB6/PB7 不是 TIMG0 输出 |
| 舵机 PWM | PA8/PA9 (TIMG2) | PA3/PA4 (TIMG8) | TIMG2 不存在于 MSPM0G3507 |
| UART TX | PA12 | PA10 | PA12 已被 TIMG0 CCP0 占用 |
| UART RX | PA11 | PA11 | 无变化 |
| I2C | PB8/PB9 (I2C0) | 暂不使用 | I2C0 固定在 PA0/PA1 |
| MODE按键 | PA13 | PB8 | PA13 已被 TIMG0 CCP1 占用 |
| START按键 | PA14 | PB11 | PA14 保留给 ADC1 (如需要) |
| 激光 | PA10 | PA20 | PA10 已被 UART0 TX 占用 |
| 编码器 | PC0-PC3 | PC0-PC3 | 无变化 |
