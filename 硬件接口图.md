## 二、MSPM0G3507 LaunchPad 引脚分配总表

### 电机驱动 (TB6612FNG)

| MSPM0 引脚 | 外设功能 | TB6612 引脚 | 说明 |
|------------|----------|-------------|------|
| **PB0** | GPIO (输出) | AIN1 | 左电机方向1 |
| **PB1** | GPIO (输出) | AIN2 | 左电机方向2 |
| **PB2** | GPIO (输出) | BIN1 | 右电机方向1 |
| **PB3** | GPIO (输出) | BIN2 | 右电机方向2 |
| **PB4** | GPIO (输出) | STBY | 使能 (HIGH=工作) |
| **PA12** | TIMG0 CCP0 | PWMA | 左电机 PWM (20kHz) |
| **PA13** | TIMG0 CCP1 | PWMB | 右电机 PWM (20kHz) |

> 注意: TIMG0 也可在 PA5/PA6 输出, 但 PA5/PA6 已分配给巡线传感器。
> 因此电机 PWM 只能走 PA12/PA13, 不可更改。

### 巡线传感器 (TCRT5000 ×7)

> **当前方案**: 数字模式 (DO)。使用 XD-01 模块 (带 LM393 比较器) 输出 TTL 电平。
> 黑线=HIGH，白底=LOW。可判断第 N 路压线，位置分辨率 ≈ 传感器间距 (1.0-1.5cm)。
>
> **如需要更高精度**: 改用模拟方案 (AO)，将传感器引脚全部换到有 ADC 功能的引脚
> (PA14-PA18, PA21-PA25 等)，通过加权平均算法获得亚毫米级位置偏差。
> 详见第七节"巡线传感器: 模拟 vs 数字方案"。

| MSPM0 引脚 | 传感器 | 说明 |
|------------|--------|------|
| **PA2** | S1 (最左) | GPIO 输入，内部上拉 |
| **PA3** | S2 | GPIO 输入 |
| **PA4** | S3 | GPIO 输入 |
| **PA5** | S4 (中间) | GPIO 输入 |
| **PA6** | S5 | GPIO 输入 |
| **PA7** | S6 | GPIO 输入 |
| **PA8** | S7 (最右) | GPIO 输入 |

> XD-01 模块自带 LM393 比较器 + 电位器调阈值，DO 输出 TTL 电平。
> 黑线=HIGH，白底=LOW。数字方案精度受限于传感器间距 (~1.2cm)，如需更高精度参见第七节。

### 舵机云台 (MG90S ×2)

| MSPM0 引脚 | 外设功能 | 舵机 | 说明 |
|------------|----------|------|------|
| **PA3** | TIMG8 CCP0 | 云台水平(X) | 50Hz PWM, 500-2500μs |
| **PA4** | TIMG8 CCP1 | 云台俯仰(Y) | 50Hz PWM, 500-2500μs |

> ⚠️ 原配置使用不存在的 TIMG2，修正为 TIMG8。
> TIMG8 是 MSPM0G3507 上可用的定时器，CCP0/CCP1 在 PA3/PA4。

**⚠️ 引脚冲突警告**: PA3/PA4 同时被分配给巡线传感器 S2/S3 和舵机 PWM。
需要在传感器和舵机之间做取舍。**建议方案：**

**方案A (推荐)**: 巡线传感器改为纯数字读取，只用 PA2, PA5-PA8, PB9, PB10 共7个GPIO，PA3/PA4 留给舵机。

**方案B**: 使用 TIMG12 (PB13/PA31) 驱动舵机，PA3/PA4 留给传感器。

> 以下按**方案A**继续分配。

## 四、电源系统设计

### 供电架构

```
2S 18650 7.4V
      │
      ├── 双路开关1 (控制回路) ──→ TB6612 VM ──→ 驱动左右电机
      │         │                      │
      │         │                 TB6612 VCC ← 3.3V (AMS1117)
      │         │
      │         └──→ AMS1117-3.3V ──→ LaunchPad 3.3V 引脚
      │                                    │
      │                    ┌───────────────┼───────────────┐
      │                    │               │               │
      │              MSPM0G3507      TCRT5000 ×7      编码器 ×2
      │              (3.3V)          (3.3V, ~140mA)   (3.3V)
      │
      └── 双路开关2 (瞄准回路) ──→ MP1584 DC-DC (7.4V→5V, 3A)
                                          │
                          ┌───────────────┼───────────────┐
                          │               │               │
                      MG90S ×2        OpenMV H7       激光模块
                    (5V, ~1.5A pk)   (5V, 250mA)     (3.3V,
                                                      LaunchPad引出,
                                                      GPIO控制)
```

> **为什么舵机和 OpenMV 不从 TB6612 板载 5V 取电？**
> WHEELTEC R3X 套件中的 TB6612 驱动板自带 5V 稳压输出，但输出电流通常 <800mA。
> 双舵机同时动作峰值可达 1.5A，加上 OpenMV (~250mA) 后峰值 ~1.8A，远超板载稳压能力，
> 会导致电压跌落、舵机抖舵、OpenMV 重启。
> **方案**: 瞄准回路用独立的 MP1584 DC-DC 降压模块 (7.4V→5V, 3A 额定) 单独供电。

### 各模块供电需求

| 模块 | 电压 | 典型电流 | 峰值电流 | 供电来源 |
|------|------|----------|----------|----------|
| MSPM0G3507 | 3.3V | 20mA | 50mA | AMS1117-3.3V (电池 7.4V → 3.3V) |
| TB6612FNG (VM) | 7.4V | — | 1.2A/通道 | 电池直连, 经开关1 |
| TT电机 ×2 | 6-7.4V | 200mA | 800mA | TB6612 输出 |
| MG90S ×2 | **5V** | 150mA | **750mA** (单只) | **MP1584 DC-DC (3A) 独立供电** |
| TCRT5000 ×7 | 3.3V | 70mA | 140mA | AMS1117-3.3V |
| OpenMV H7 | **5V** | 150mA | 250mA | **MP1584 DC-DC 5V** |
| 激光模块 | 3.3V | 20mA | 50mA | AMS1117-3.3V (或模块自带 3.3V) |
| 编码器 ×2 | 3.3V | 10mA | 20mA | AMS1117-3.3V |
| **合计** | | **~570mA** | **~3.5A** | |

> **电池建议**: 2S 18650 7.4V, 2600mAh 以上 (松下 3400mAh 或三星 2600mAh)。
> 18650 锂离子电池不受北京航空件限制，比 LiPo 更方便运输。

### MP1584 DC-DC 模块配置

| 参数 | 设定 |
|------|------|
| 输入 | 7.4V (2S 电池, 经开关2) |
| 输出 | **5.0V** |
| 额定电流 | 3A |
| 用途 | MG90S ×2 + OpenMV H7 |
| 滤波 | 输入/输出端各并 100μF 电解 + 0.1μF 陶瓷 |

> 调节 MP1584 模块上的蓝色电位器，用万用表确认输出为 5.0V ±0.1V 后再接入负载。
> 模块先调好电压，再接舵机和 OpenMV，避免过压烧毁。

## 五、TB6612FNG 接线详图

```
                    TB6612FNG 模块
                 ┌──────────────────┐
    VM (7.4V) ───┤ VM          AO1 ├──→ 左电机 +
                 │                  │
    GND ─────────┤ GND         AO2 ├──→ 左电机 -
                 │                  │
    PB0 ─────────┤ AIN1        BO1 ├──→ 右电机 +
                 │                  │
    PB1 ─────────┤ AIN2        BO2 ├──→ 右电机 -
                 │                  │
    PB2 ─────────┤ BIN1           │
                 │                  │
    PB3 ─────────┤ BIN2    5V OUT ├──→ (不使用, 舵机/OpenMV 由 MP1584 独立供电)
                 │                  │
    PA12 ────────┤ PWMA           │
                 │                  │
    PA13 ────────┤ PWMB           │
                 │                  │
    PB4 ─────────┤ STBY           │
                 └──────────────────┘

真值表:
  AIN1=H, AIN2=L, PWM>0 → 左电机正转
  AIN1=L, AIN2=H, PWM>0 → 左电机反转
  AIN1=H, AIN2=H, PWM>0 → 刹车
  STBY=L → 全部停止
```

## 六、分步上电验证流程 (防烧毁)

> ⚠️ 严格按照顺序操作，每步确认后再进行下一步。

### 第0步: 目视检查 + 电源预调 (不通电)

- [ ] 检查所有焊点有无虚焊、短路
- [ ] 用万用表测量电池电压 (应为 7.4-8.4V)
- [ ] 用万用表测量各模块 VCC-GND 之间有无短路

### 第1步: LaunchPad 单独上电

- [ ] USB 连接电脑，确认 CCS 能识别设备
- [ ] 下载一个 LED 闪烁测试程序
- [ ] 确认 3.3V 和 5V 引脚电压正常

### 第2步: 传感器模块测试 (不接电机)

- [ ] 7 个 TCRT5000 模块接 3.3V 供电
- [ ] **模拟输出模块 (BOM推荐)**: 用万用表测量 AO 引脚电压, 白纸上应 >2V (高反射), 黑线上应 <1V (低反射), 确认黑白对比度足够

- [ ] 下载测试程序读取 7 路 GPIO/ADC 状态，串口打印

### 第3步: 舵机测试 (不接电机)

- [ ] MG90S ×2 接 **MP1584 5V 输出** (已预调至 5.0V), 不经过 LaunchPad
- [ ] MG90S GND 与 LaunchPad GND 共地
- [ ] 信号线接 PA3/PA4
- [ ] 下载测试程序: 输出 1500μs (中位)，确认舵机归中
- [ ] 测试 500-2500μs 范围，确认舵机转动平滑无抖舵
- [ ] 双舵机同时快速摆动，用万用表监测 MP1584 输出，确认 5V 不跌落 >0.3V

### 第4步: 电机驱动测试 (不装轮胎)

- [ ] TB6612 接电池 7.4V (VM)
- [ ] STBY 接 HIGH
- [ ] 先不接 PWM，只测试方向引脚: AIN1=H, AIN2=L → 电机应转动
- [ ] 接 PWM (PA12)，从低占空比 (10%) 开始，逐步增大
- [ ] 确认两个电机都能正反转
- [ ] 测试刹车 (AIN1=H, AIN2=H)

### 第5步: 编码器测试

- [ ] **推荐方法 (无需示波器)**: 下载测试程序, GPIO 双沿中断计数, 手转一圈, 串口打印脉冲数
- [ ] WHEELTEC R3X (500线 GMR, 1:30减速): 手转输出轴一圈 → 应计 ~60000 边沿 (500×30×4倍频)
- [ ] 如只有万用表: A/B 相分别对 GND 测量, 慢转电机时应看到电压在 0V 和 3.3V 之间交替变化
- [ ] 正转时 A 相应超前 B 相 90°, 确认方向判断正确

### 第6步: OpenMV 通信测试

- [ ] OpenMV 接 MP1584 5V 供电
- [ ] **重要**: 拔掉 LaunchPad 上 J21(TX) 和 J22(RX) 跳线帽，断开 XDS110 调试器与 UART0 的连接，避免调试器干扰 OpenMV 通信帧
- [ ] OpenMV P4(TX) → PA11 (MSPM0 RX), OpenMV P5(RX) → PA10 (MSPM0 TX)
- [ ] OpenMV 连接电脑 USB 下载颜色追踪脚本, 下载完成后拔掉 USB 只保留 5V 供电
- [ ] 下载 MSPM0 测试程序: 发送请求帧, 接收坐标数据, 串口打印 (调试用 SWO 或 LED 闪烁, 不用 UART0)

- [ ] PA20 输出 HIGH，确认激光亮
- [ ] 确认光斑直径 ≤0.5cm (调焦)

### 第8步: 整机联调

- [ ] 装轮胎，放在赛道上
- [ ] 测试巡线 (基本要求1)
- [ ] 测试静态瞄准 (基本要求2/3)
- [ ] 测试行进中瞄准 (发挥部分)

## 七、硬件注意事项与保护电路

### 7.1 巡线传感器: 模拟 vs 数字方案

BOM 推荐购买**分立 TCRT5000 模拟输出 (AO)** 模块，可通过加权平均算法获得亚毫米级位置精度。
但方案 A 选用的 7 个 GPIO (PA2, PA5-PA8, PB9-PB10) **均无 ADC 功能**，只能读取数字信号。

| | 模拟方案 | 数字方案 (当前) |
|------|---------|-------------|
| 精度 | 亚毫米 (加权平均) | ~1.2cm (仅知第几路压线) |
| ADC 需求 | 7 通道 | 0 |
| 模块 | 分立 TCRT5000 AO | XD-01 带 LM393 比较器 |
| 引脚 | 需改用 ADC 引脚 (PA14-PA25) | 当前 GPIO 即可 |

**建议**: 如需使用模拟方案，将巡线传感器全部改接到有 ADC 功能的引脚 (PA14-PA18, PA21-PA25 等)，
使用 `adc12_sequence_conversion` 循环采集 7 通道。

### 7.2 UART0 调试冲突

PA10/PA11 同时连接了 OpenMV 和 XDS110 调试器虚拟串口。开发时必须注意:

- **开发调试时**: 拔掉 LaunchPad 上 J21(TX) 和 J22(RX) 跳线帽，断开调试器，OpenMV 独占 UART0
- **需要 printf 调试时**: 插回跳线帽，断开 OpenMV 的 TX/RX 连接
- **更优方案**: 用 UART1 (PB4=TX, PB5=RX) 接 OpenMV，UART0 保留给调试 printf

### 7.3 保护电路

```
电池 7.4V+ ──┬── SS14 肖特基二极管 (防反接) ──→ 双路开关
             │    (阳极朝电池+, 阴极朝负载)
             │
电池 7.4V- ──┴── GND

电机端:
  TB6612 AO1/AO2 之间并 0.1μF 陶瓷电容 (吸收换向火花)
  TB6612 BO1/BO2 之间并 0.1μF 陶瓷电容
```

| 保护措施 | 位置 | 说明 |
|---------|------|------|
| SS14 肖特基二极管 | 电池输出端 | 反接时截止, 防止烧毁 TB6612/AMS1117 |
| 1000μF/16V 电解 ×2 | TB6612 VM 端 + MP1584 输出端 | 吸收电流尖峰 |
| 0.1μF 陶瓷电容 ×N | **每个模块** VCC-GND 旁 | 高频去耦, 紧贴模块电源引脚焊接 |
| 100μF 电解 ×2 | AMS1117 输入/输出端 | 低频滤波 |

> SS14 (1A 40V) 约 0.2 元/只，串在电池正极输出端。如果电池接头本身有防反插设计 (如 XT30 接头)，可省略。

### 7.4 MG90S 信号电平

MSPM0 输出 3.3V PWM，MG90S 供电 5V。实测绝大部分 MG90S 接受 3.3V 逻辑电平 (TTL 阈值 ~1.5V)。
首次测试时若舵机不响应或抖动，在 PA3/PA4 与舵机信号线之间加 74HC125 电平转换 (3.3V→5V)。

### 7.5 编码器引脚确认

PC0-PC3 在 MSPM0G3507 64-LQFP 封装中存在，但在 LP-MSPM0G3507 LaunchPad 的 BoosterPack 引脚上
不一定全部引出。焊接前对照 [LP-MSPM0G3507 原理图](https://www.ti.com/lit/pdf/slau938)
确认 PC0-PC3 在 J1-J4 排针上的实际位置。

### 7.6 共地要求

所有模块的 GND 必须连通 (共地):
- LaunchPad GND ↔ TB6612 GND ↔ MP1584 GND ↔ MG90S GND ↔ OpenMV GND
- 电池负极 → 所有 GND
- 功率地 (电机/舵机) 和信号地 (传感器/OpenMV) 在电池负极处**单点汇合**，减少电机噪声干扰传感器
