# 两轮差速驱动 — 运动学模型

## 符号定义

| 符号 | 含义 | 单位 |
|------|------|------|
| v_l, v_r | 左/右轮线速度 | m/s |
| v_c | 车体中心线速度 | m/s |
| omega_c | 车体角速度 | rad/s |
| L | 两轮轮间距 (轴距) | m |
| d | 车轮直径 | m |
| R | 转弯半径 | m |
| theta | 车体航向角 | rad |

## 正运动学 (轮速 → 车体速度)

```
v_c   = (v_r + v_l) / 2
omega = (v_r - v_l) / L
R     = v_c / omega = (L/2) * (v_r + v_l) / (v_r - v_l)
```

特殊情况:
- 直线: omega=0, v_r=v_l → R=∞
- 原地旋转: v_c=0, v_r=-v_l → R=0
- 左转: v_r > v_l → omega>0 → 圆心在车体左侧

## 逆运动学 (期望速度 → 轮速)

```
v_l = v_c - omega * L / 2
v_r = v_c + omega * L / 2

转速(rpm) = v / (PI * d) * 60
```

## 里程计 (编码器 → 位姿推算)

由编码器读数得到左右轮行驶弧长 delta_Sl, delta_Sr:

```
delta_theta = (delta_Sr - delta_Sl) / L
delta_S     = (delta_Sr + delta_Sl) / 2

delta_x     = delta_S * cos(theta + delta_theta/2)
delta_y     = delta_S * sin(theta + delta_theta/2)

x     += delta_x
y     += delta_y
theta += delta_theta
```

| 轮间距 L | 150mm (0.15m) |
| 轨迹周长 | 4m |
| 要求速度 | ≥ 4m/20s = 0.2 m/s |
