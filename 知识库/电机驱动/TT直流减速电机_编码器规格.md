## 编码器规格

| 参数 | 数值 |
|------|------|
| 类型 | 霍尔传感器 (AB相正交) |
| 工作电压 | 3.3V ~ 5V |
| 线数 | 8~60线/转 (取决于型号) |
| 信号类型 | 正交方波 (A/B相) |
| 每转脉冲 (4倍频) | 线数 × 4 (如13线: 52脉冲/转) |

### 编码器接线 (6P)

| 引脚 | 功能 | 颜色 |
|------|------|------|
| M- | 电机负极 | 黑 |
| M+ | 电机正极 | 红 |
| Vcc | 编码器供电 | 红/白 |
| H1 (A) | A相输出 | 黄 |
| H2 (B) | B相输出 | 绿 |
| GND | 编码器地 | 黑/蓝 |

## 转速计算

```
// M法测速
#define PULSE_PER_REV   13      // 编码器线数
#define REDUCTION_RATIO 20      // 减速比
#define SAMPLING_MS     20      // 采样周期

// 4倍频后每转脉冲数
#define PULSE_TOTAL (PULSE_PER_REV * REDUCTION_RATIO * 4)

// RPM = (pulse_count / PULSE_TOTAL) * (60000 / SAMPLING_MS)
```
