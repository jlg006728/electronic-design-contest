# TB6612FNG 数据手册摘要

> 来源: Toshiba, [Pololu #713](https://www.pololu.com/product/0713/resources)

## 芯片概述

东芝双 H 桥电机驱动 IC，LD MOS 结构，低导通电阻。可驱动 2 路直流电机或 1 路双极性步进电机。

## 关键规格

| 参数 | 数值 |
|------|------|
| 电机电源 VM | 2.5 – 13.5 V (最大 15V) |
| 逻辑电源 VCC | 2.7 – 5.5 V (最大 6V) |
| 连续电流/通道 | **1.0 A** (VM ≥ 4.5V) |
| 平均电流/通道 | 1.2 A |
| 峰值电流 (10ms) | **3.2 A** |
| 导通电阻 | **0.5 Ω** typ |
| PWM 频率 | 最高 100 kHz |
| 待机电流 | ≤ 1 μA |
| 封装 | SSOP-24 |
| 工作温度 | -20°C ~ +85°C |

## 引脚说明

| 引脚 | I/O | 功能 |
|------|-----|------|
| AIN1/AIN2 | I | A通道方向 (200kΩ下拉) |
| PWMA | I | A通道 PWM (200kΩ下拉) |
| BIN1/BIN2 | I | B通道方向 (200kΩ下拉) |
| PWMB | I | B通道 PWM (200kΩ下拉) |
| STBY | I | 待机 (LOW=待机) |
| AO1/AO2 | O | A通道电机输出 |
| BO1/BO2 | O | B通道电机输出 |
| VM1-VM3 | — | 电机电源 |
| VCC | — | 逻辑电源 |
| PGND | — | 功率地 |

## 控制真值表

| IN1 | IN2 | PWM | STBY | 模式 |
|-----|-----|-----|------|------|
| H | H | H/L | H | **短路刹车** |
| L | H | H | H | **反转 (CCW)** |
| H | L | H | H | **正转 (CW)** |
| L | L | H | H | **滑行停止** |
| X | X | X | L | **待机** |

## 内置保护

- 过热关断: ~175°C (滞后 20°C)
- 欠压锁定: 检测 ~1.9V, 恢复 ~2.2V
- 死区插入: 50ns (H→L) / 230ns (L→H)
- 内置续流二极管
