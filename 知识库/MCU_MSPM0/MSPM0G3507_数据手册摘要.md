# MSPM0G3507 数据手册摘要

> 来源: TI官网 [ti.com/product/MSPM0G3507](https://www.ti.com/product/MSPM0G3507)
> 文档版本: Rev. C (2025年10月)

## 芯片概述

MSPM0G3507 属于 TI MSPM0 G 系列，混合信号 Arm Cortex-M0+ MCU，专为电机控制和传感测量设计。

## 核心规格

| 参数 | 数值 |
|------|------|
| CPU | Arm Cortex-M0+ @ **80 MHz** (带 MPU) |
| Flash | **128 KB** (带 ECC 纠错) |
| SRAM | **32 KB** (带硬件奇偶校验) |
| 供电电压 | 1.62 V – 3.6 V |
| 温度范围 | –40°C 至 +125°C |
| RUN 功耗 | 96 μA/MHz |
| STANDBY 功耗 | 1.5 μA |
| SHUTDOWN 功耗 | 80 nA |

## 模拟外设

| 外设 | 数量 | 关键指标 |
|------|------|----------|
| ADC | 2× 12-bit | **4 Msps**, 最多 17 外部通道 |
| DAC | 1× 12-bit | 1 Msps |
| OPA | 2× | 零漂移斩波, 增益 1-32x |
| COMP | 3× | **32 ns** 传播延迟 |

## 数字外设

| 外设 | 数量 | 关键指标 |
|------|------|----------|
| CAN-FD | 1× | CAN 2.0 A/B + CAN-FD |
| UART | 4× | |
| I²C | 2× | FM+ |
| SPI | 2× | 最高 32 Mbit/s |
| 定时器 | 7 个 | 最多 **22 路 PWM** |
| DMA | 7 通道 | |

## 安全与加速

- 数学加速器: DIV, SQRT, MAC, TRIG 硬件运算
- AES-128/256, CRC-16/32, 真随机数发生器 (TRNG)

## 封装

64-LQFP / 48-LQFP / 32-VQFN / 28-VSSOP

## 开发资源

- LaunchPad: LP-MSPM0G3507
- SDK: MSP SDK (随 Code Composer Studio 免费提供)
- TRM: MSPM0 G-Series 80-MHz TRM (Rev. D, 2025年3月)

## E题适配分析

- 22 路 PWM: 可同时控制双路电机 + 2路舵机
- 4 Msps ADC: 满足高速巡迹传感器阵列采样
- 内置 OPA: 可简化传感器信号调理
- 80 MHz: 足够实时控制 + 简单 PID
