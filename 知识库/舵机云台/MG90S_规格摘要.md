# MG90S / SG90 舵机规格摘要

## 概述

MG90S 是 SG90 的金属齿轮升级版，广泛用于航模和小型云台。E题需要 2 个舵机构建二维云台控制激光笔方向。

## 规格对比

| 参数 | SG90 | MG90S (推荐) |
|------|------|-------------|
| 重量 | 9 g | 13.4 g |
| 尺寸 (mm) | 21.5×11.8×22.7 | 22.8×12.2×28.5 |
| 电压 | 3.0-7.2 V | 4.8-6.0 V |
| 扭矩 (4.8V) | 1.2 kg·cm | **1.8 kg·cm** |
| 扭矩 (6.0V) | 1.8 kg·cm | **2.2 kg·cm** |
| 速度 (4.8V) | 0.12 s/60° | 0.10 s/60° |
| 速度 (6.0V) | 0.10 s/60° | 0.08 s/60° |
| 转角 | 90°/180° | 90°/180° |
| 死区 | 5-7 μs | 5 μs |
| 齿轮 | 尼龙 | **金属** |
| 轴承 | 单球 | **双球** |
| 电机 | 有刷 | **无芯** |

## PWM 控制协议

- 频率: **50 Hz** (周期 20 ms)
- 脉宽范围: **500 – 2500 μs**
- 角度映射: 500μs=0°, 1500μs=90°, 2500μs=180°



## 接线

| 颜色 | 功能 |
|------|------|
| 棕色 | GND |
| 红色 | VCC (+5V) |
| 橙色 | PWM 信号 (3.3V 兼容) |

## E题云台精度估算

- MG90S 死区 ~5 μs → 角分辨率 ≈ 0.4°
- 靶距 50cm: 光斑位移 ≈ 50cm × tan(0.4°) ≈ 3.5 mm
- 题目要求 D₁ ≤ 2cm → 精度可满足

## 供电注意事项

- 双舵机峰值电流可达 1.5A
- 建议独立 5V/2A 供电, 不与 MCU 共源
- 舵机电源端并联 1000μF 电解电容
