# 405nm 蓝紫激光模块与紫外感光纸

## 题目要求

- 激光笔: **405nm 波长**蓝紫光
- 光功率: **≤10mW**
- 光斑直径: **≤0.5cm** (需可调焦)
- 紫外感光纸: A4 幅面, 白色, 可重复使用

## 405nm 激光模块规格

| 参数 | 规格 |
|------|------|
| 波长 | 405 nm (蓝紫光) |
| 光功率 | ≤10 mW (题目限制) |
| 供电电压 | 3.0V – 5.0V DC |
| 工作电流 | 20–40 mA (取决于功率) |
| 光斑直径 | 可调 (聚焦后 <0.5cm @ 50cm) |
| 外形尺寸 | 12×30mm (典型) |
| 散热 | 小功率无需额外散热 |

### 调焦方法

```
激光管 → 聚焦透镜 → 准直透镜 → 出光口
         [可旋转调节]

调节步骤:
1. 目标靶距 50cm 处放置白纸
2. 缓慢旋转透镜调节光斑大小
3. 直到光斑最小最亮 (直径 ≤0.5cm)
4. 锁定透镜位置
```

## 紫外感光纸特性

| 参数 | 说明 |
|------|------|
| 幅面 | A4 (210mm × 297mm) |
| 颜色 | 白色 |
| 感光波长 | 紫外/近紫外 (350-410nm) |
| 痕迹颜色 | 紫色/蓝色 (光照后变色) |
| 痕迹滞留 | **20s ~ 60s** |
| 可重复使用 | 是 (痕迹消失后可重新照射) |
| 灵敏度 | 随照射强度增加, 痕迹越深越持久 |

### 比赛影响

- 痕迹会在 20-60s 内自然消失 → 评分时需快速测量
- 功率越高, 痕迹越深越久
- 弱光环境有助于观察光斑位置
- 可准备备用的感光纸

## 靶面参数

| 参数 | 规格 |
|------|------|
| 靶纸材质 | A4 UV 紫外感光纸 |
| 靶心 | 红色油性记号笔点, 直径 **≤0.1cm** |
| 同心圆 | 半径 2, 4, 6, 8, 10 cm |
| 圆弧线宽 | **≤0.1cm** |
| 推荐画笔 | 红色油性记号笔 (不溶于水) |

### 靶面制作建议

```
1. 在 A4 感光纸上用圆规画同心圆
2. 用红色油性细记号笔描线 (线宽 <0.1cm)
3. 圆心处用力点一个红点
4. 圆规针尖孔处用红色填充
```

## 激光安全注意事项

- 405nm 属可见光, 但接近紫外波段
- 禁止直射眼睛 (可能导致视网膜损伤)
- 建议佩戴 UV 防护眼镜
- 比赛期间注意周围人员安全
