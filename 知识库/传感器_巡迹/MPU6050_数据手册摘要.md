# MPU6050 数据手册摘要

> 六轴传感器: 三轴加速度计 + 三轴陀螺仪
> 厂商: TDK InvenSense | 接口: I2C (400kHz)

## 基本规格

| 参数 | 数值 |
|------|------|
| 封装 | QFN-24 (4x4x0.9mm) |
| 工作电压 | 2.375V ~ 3.46V (典型 3.3V) |
| I2C 地址 | **0x68** (AD0=0) / **0x69** (AD0=1) |
| ADC 分辨率 | 16 位 |
| FIFO | 1024 字节 |
| 待机电流 | 5μA |
| 工作温度 | -40°C ~ +85°C |

## 加速度计

| 量程 | 灵敏度 |
|------|--------|
| ±2g | 16384 LSB/g |
| ±4g | 8192 LSB/g |
| ±8g | 4096 LSB/g |
| ±16g | 2048 LSB/g |

## 陀螺仪

| 量程 | 灵敏度 |
|------|--------|
| ±250°/s | 131 LSB/(°/s) |
| ±500°/s | 65.5 LSB/(°/s) |
| ±1000°/s | 32.8 LSB/(°/s) |
| ±2000°/s | 16.4 LSB/(°/s) |

## 关键寄存器

| 名称 | 地址 | 功能 |
|------|------|------|
| PWR_MGMT_1 | 0x6B | 复位(bit7)、睡眠(bit6)、时钟源 |
| SMPLRT_DIV | 0x19 | 采样率除频 |
| CONFIG | 0x1A | DLPF 低通滤波 |
| GYRO_CONFIG | 0x1B | 陀螺仪量程 |
| ACCEL_CONFIG | 0x1C | 加速度计量程 |
| ACCEL_XOUT_H | 0x3B | 加速度数据起始 (连续读14B) |
| GYRO_XOUT_H | 0x43 | 陀螺仪数据起始 |
| WHO_AM_I | 0x75 | 设备ID (0x68) |

## 初始化流程

```c
// 1. 复位 → 2. 唤醒 → 3. 采样率 → 4. DLPF → 5. 配置量程
MPU_WriteReg(0x6B, 0x80); delay(100);  // 复位
MPU_WriteReg(0x6B, 0x01);              // 唤醒, X轴PLL
MPU_WriteReg(0x19, 0x07);              // 1kHz/(1+7)=125Hz
MPU_WriteReg(0x1A, 0x03);              // DLPF 44Hz
MPU_WriteReg(0x1B, 0x18);              // 陀螺 ±2000°/s
MPU_WriteReg(0x1C, 0x00);              // 加计 ±2g
```

## E题应用: 航向保持 (Z轴转角)

```
陀螺仪 Z轴 → 数值积分 → 航向角 (yaw)
          → 漂移校准 (取静止100帧均值作偏移量)
          → 转向PID: target_yaw vs current_yaw

转向时:
- 设定目标航向 (90° 转弯)
- 实时读取 current_yaw (陀螺仪积分)
- gyro_pid.update(target_yaw, current_yaw) → 差速PWM
```

## 注意事项

| 事项 | 说明 |
|------|------|
| 上拉电阻 | SDA/SCL 各 4.7kΩ → 3.3V |
| 零偏校准 | 静止时采集100帧取均值作为偏移 |
| 振动干扰 | 固定牢靠, 远离电机振动 |
| 温漂 | 温差大时需重校零偏 |
