# PID控制理论与巡迹算法参考

## PID算法对比

### 位置式PID

```
u(k) = Kp*e(k) + Ki*SUM[e(j)] + Kd*[e(k) - e(k-1)]
```

输出为全量控制值。适用于**转向环**（直接输出目标角度/占空比）。

### 增量式PID

```
delta_u(k) = Kp*[e(k)-e(k-1)] + Ki*e(k) + Kd*[e(k)-2*e(k-1)+e(k-2)]
```

输出为控制增量。适用于**速度环**（叠加到当前PWM）。

## E题控制架构

```
传感器阵列 (7路TCRT5000)
    │
    ▼  位置偏差 e_pos
┌──────────┐
│ 位置外环  │ 位置式PD (Kp_outer, Kd_outer)
└────┬─────┘
     │  目标角速度 / 左右轮速度偏置
     ▼
┌──────────┐
│ 速度内环  │ 增量式PI (Kp_inner, Ki_inner)
└────┬─────┘
     │  PWM增量
     ▼
┌──────────┐
│ TB6612   │ ─── 左轮 + 右轮
└──────────┘

反馈:
  OpenMV ── 靶心坐标偏差 ──→ 云台PID (位置式PD) ──→ 舵机角度
  编码器 ── 轮速 ──→ 速度环PI
```

## PID参数整定口诀

> 参数整定找最佳，从小到大顺序查。
> 先是比例后积分，最后再把微分加。
> 曲线振荡很频繁，比例度盘要放大。
> 曲线漂浮绕大弯，比例度盘往小扳。
> 曲线偏离回复慢，积分时间往下降。
> 曲线波动周期长，积分时间再加长。
> 曲线振荡频率快，先把微分降下来。
> 动差大来波动慢，微分时间应加长。
> 理想曲线两个波，前高后低四比一。

## 实用PID技巧

| 技巧 | 说明 |
|------|------|
| **积分限幅** | 防止积分饱和 (I_max = ±30% PWM) |
| **积分分离** | 大偏差(|e|>阈值)时关闭积分, 小偏差时开启 |
| **微分先行** | 对测量量微分而非偏差量, 避免设定突变冲击 |
| **死区** | |e|<死区时不调节, 减少抖动 |
| **输出限幅** | PWM范围约束 [-100, 100] |

## 传感器阵列位置偏差计算

使用 7 路传感器, 间距 1.2cm:

```
传感器编号:  -3  -2  -1   0  +1  +2  +3
物理位置:   左3 左2 左1  中  右1 右2 右3

偏差计算 (加权平均):
e_pos = SUM(i * (1 - sensor[i])) / SUM(1 - sensor[i])
其中 sensor[i]=0 表示检测到黑线

若仅有编号i传感器检测到黑线:
e_pos = i * 1.2cm (物理偏差, 单位cm)
```

## 典型参数初值

| 控制环 | Kp | Ki | Kd |
|--------|------|------|------|
| 位置环 (巡迹) | 0.5~2.0 | (通常不加) | 0.1~0.5 |
| 速度环 | 0.3~1.0 | 0.05~0.3 | (通常不加) |
| 云台水平 | 0.02~0.1 | 0.001~0.01 | 0.005~0.02 |
| 云台俯仰 | 0.02~0.1 | 0.001~0.01 | 0.005~0.02 |
