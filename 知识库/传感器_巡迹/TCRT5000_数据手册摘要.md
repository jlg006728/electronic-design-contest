# TCRT5000 数据手册摘要

> 来源: Vishay Semiconductors ([vishay.com/docs/83760](https://www.vishay.com/docs/83760/tcrt5000.pdf))

## 器件概述

反射式光电传感器，红外发射管 + 光敏三极管一体封装，带环境光阻挡滤光片。

## 封装参数

| 参数 | 数值 |
|------|------|
| 封装 | 直插式 (through-hole) |
| 尺寸 | 10.2 × 5.8 × 7.0 mm |
| 峰值工作距离 | 2.5 mm |
| 工作范围 | 0.2 – 15 mm |

## 绝对最大额定值 (25°C)

| 参数 | 发射管 (LED) | 接收管 (三极管) |
|------|-------------|----------------|
| 反向电压 | 5 V | — |
| 正向电流 | 60 mA | — |
| C-E 电压 | — | 70 V |
| 集电极电流 | — | 100 mA |
| 功耗 | 100 mW | 100 mW |

## 关键电学特性 (25°C)

| 参数 | 条件 | 典型值 |
|------|------|--------|
| 正向电压 V_F | I_F = 60 mA | 1.25 V |
| 峰值波长 λ_P | — | 940 nm |
| 集电极电流 I_C | V_CE=5V, I_F=10mA, d=12mm | 1 mA |
| 暗电流 | V_CE=20V | ≤ 200 nA |

## E题巡迹应用电路

```
VCC (3.3V) ─┬─ R1(100Ω) ── LED发射管 ── GND
             │
             └─ R2(10kΩ上拉) ── OUT ──→ MSPM0 PA0~PA6 (ADC)
                        │
                  光敏三极管集电极
                        │
                       GND
```

**7路传感器阵列排列** (间距 ~1.2cm):
```
  [S1] [S2] [S3] [S4] [S5] [S6] [S7]
   │    │    │    │    │    │    │
  PA0  PA1  PA2  PA3  PA4  PA5  PA6  → ADC12 采样
```

**工作原理**:
- 黑线吸收红外 → 接收管截止 → 输出 HIGH
- 白底反射红外 → 接收管导通 → 输出 LOW

## E题配置建议

| 参数 | 建议值 |
|------|--------|
| 传感器数量 | 7-8 路 |
| 传感器间距 | 1.0-1.5 cm |
| 离地高度 | 3-5 mm |
| 采样频率 | ≥ 100 Hz |
| 发射管限流 | 100-220Ω |
| 接收管上拉 | 10-47kΩ |

## 引脚定义 (模块)

| 引脚 | 功能 |
|------|------|
| VCC | 3.3V / 5V |
| GND | 地 |
| OUT | 数字输出 (可设阈值) / 模拟输出 |
