# 电赛设计报告模板

> TI杯全国大学生电子设计竞赛 — 设计总结报告

## 评分标准 (报告占 ~30分)

| 评分项 | 分值参考 | 要点 |
|--------|----------|------|
| 方案设计与论证 | 6分 | 2-3个方案, 对比分析, 选定理由 |
| 理论分析与计算 | 6分 | 电路原理, 参数工程计算 |
| 电路图及设计文件 | 6分 | 完整、规范、国标符号 |
| 测试方法与数据 | 8分 | 仪器型号, 原始数据, 完整性 |
| 结果分析与讨论 | 4+5~10分 | 误差分析, 改进方向, 创新点 |
| 卷面工整性 | 2~3分 | 排版规范, 图表清晰 |

## 报告结构模板

### 封面
- 题号 + 题目名称 (不出现学校/姓名/教师)

### 摘要 (200-300字)
- 目的 → 方法 → 结果 → 结论
- 中英文各一份, 关键词 3-8个

### 目录
- 列出3级标题及页码

### 正文 (6页以内, ~5000字)

```
1 系统方案比较与设计
  1.1 方案1: xxx
  1.2 方案2: xxx
  1.3 方案3: xxx
  1.4 方案论证与选定

2 理论分析与计算
  (核心电路原理 + 工程公式计算)

3 单元电路设计与计算
  3.1 巡迹传感器电路
  3.2 电机驱动电路
  3.3 云台控制电路
  3.4 视觉模块接口
  ...

4 程序设计
  4.1 程序流程图
  4.2 关键算法说明 (PID, 巡迹, 瞄准)

5 系统测试
  5.1 测试仪器 (名称、型号、精度)
  5.2 测试方法与步骤
  5.3 测试数据与结果
  5.4 误差分析

6 结论
  (达标情况、优缺点、改进方向)

7 参考文献
```

### 附录
- 系统操作说明
- 元器件明细表
- 仪器设备清单
- 电原理图 (可另附)
- PCB图
- 程序清单

## 格式规范

| 项目 | 规范 |
|------|------|
| 纸张 | A4 纵向 |
| 正文篇幅 | 限6页 (电路图可另附) |
| 正文字体 | 小四号宋体, 单倍行距 |
| 页眉 | 上方≥3cm空白 |
| 页码 | 每页右下角 |
| 一级标题 | 小二号黑体, 居中 |
| 二级标题 | 三号标宋, 居中 |
| 三级标题 | 四号黑体, 顶格 |
| 英文/数字 | Times New Roman |
| 图序图题 | 小五号宋体, 图下方居中 |
| 表序表题 | 小五号黑体, 表上方居中 |

## 写作要点

- 方案论证必须有3个方案, 缺一个扣2分
- 电路图用新国标符号, 不看正文也能理解
- 测试数据要原始完整, 标注仪器型号精度
- 特色/创新单独给分 5-10分, 是拉开差距关键
- 不出现学校名称、参赛者姓名、指导教师
