# E题供电方案设计

## 题目供电要求

- MSPM0 控制器和瞄准模块需 **两个独立开关** 分别控制
- 开关须有 **发光管** 指示供电状态
- 车载电池供电, 行驶中不得人为干涉或遥控

## 系统供电架构

│  │ 锂电池       │    │ 锂电池        │           │
│  └──┬───────────┘    └──┬───────────┘           │

│  │ 开关1 + │        │ 开关2 + │                │

│  └──┬──────┘        └──┬──────┘                │
│     │                   │                       │

## 各路电压需求分析

### 开关1: MSPM0 控制回路

| 负载 | 电压 | 峰值电流 |
|------|------|----------|
| MSPM0G3507 | 3.3V | ~50mA |
| TCRT5000 x8 | 3.3V | ~160mA (20mA×8) |
| TB6612 逻辑 | 3.3V | ~10mA |
| TT电机 x2 | 7.4V | ~2A (启动/堵转) |
| 编码器 x2 | 3.3V | ~20mA |
| LED指示 | 3.3V | ~5mA |
| **总计** | — | **~2.2A 峰值** |

### 开关2: 瞄准模块回路

| 负载 | 电压 | 峰值电流 |
|------|------|----------|
| OpenMV | 5V | ~250mA |
| 舵机 x2 | 5V | ~1.5A (同时动作) |
| 激光笔 | 3.3V | ~40mA |
| LED指示 | 5V | ~5mA |
| **总计** | — | **~1.8A 峰值** |

## 推荐电池方案

### 方案A: 双电池独立供电 (推荐)

| 子系统 | 电池 | 容量建议 | 续航估算 |
|--------|------|----------|----------|
| 控制回路 | 2S 7.4V 锂电池 | 2000mAh | ~2h |
| 瞄准模块 | 5V 锂电池 (充电宝型) | 3000mAh | ~3h |

### 方案B: 单电池 + 双路稳压

| 电池 | 稳压 |
|------|------|
| 3S 11.1V 锂电 | LDO #1: 5V/2A → 瞄准模块 |
| 容量: 3000mAh+ | LDO #2: 7.4V/2A → 电机 + 3.3V LDO → MCU |

## 稳压芯片推荐

| 芯片 | 输入 | 输出 | 电流 | 用途 |
|------|------|------|------|------|
| AMS1117-3.3 | 4.5-12V | 3.3V | 1A | MCU, 传感器 |
| MP1584 | 4.5-28V | 可调 | 3A | DC-DC 降压 (电机) |
| LM2596 | 4.5-40V | 可调/固定 | 3A | DC-DC 降压 |
| 7805 | 7-35V | 5V | 1.5A | 舵机供电 (简单) |

## 开关与指示电路

```
电池+ ──┬── 拨动开关 ──┬────────── 负载
        │              │
        ├── 1kΩ ─── LED (红色) ── GND
        │              ↑ 开关接通时亮

电池+ ──┬── 拨动开关 ──┬────────── 负载
        │              │
        ├── 1kΩ ─── LED (蓝色) ── GND
        │              ↑ 开关接通时亮
```

## E题供电注意事项

- 两个开关必须分别控制, 不可串联
- LED 指示必须直观可见 (建议不同颜色)
- 电池须固定牢靠, 避免行驶中脱落
- 建议并联大电容 (1000μF) 在电机和舵机电源端
- 信号地 (MCU) 和功率地 (电机) 须单点接地避免干扰
