# 接线步骤 — 一步一步来

> 按模块逐步接线，每步完成后检查再进入下一步。
> 所有接线操作在**断电状态**下进行。

---

## 准备工作

**工具清单**:
- 万用表 (测电压/通断)
- 杜邦线 (公对母、母对母各若干)
- 电烙铁 + 焊锡 (如需焊接)
- 热缩管 / 电工胶带
- 螺丝刀 (固定底盘/模块)

**先认清 LaunchPad 排针**:

## 第1步：电源模块接线

**目标**: 搭建双路供电系统，不接任何负载。

### 1.1 MP1584 DC-DC 降压模块预调

```
电池 7.4V+ ──→ MP1584 IN+
电池 7.4V- ──→ MP1584 IN-
```

- 用万用表测量电池电压 (应为 7.4-8.4V)
- 临时接通电池，用万用表测 MP1584 输出端
- 调节蓝色电位器，使输出为 **5.0V ±0.1V**
- 调好后**断开电池**

### 1.2 双路开关接线

```
电池 7.4V+ ──→ 开关1 公共端 ──→ 开关1 ON ──→ TB6612 VM+ (电机电源)
                │
                └──→ 开关2 公共端 ──→ 开关2 ON ──→ MP1584 IN+ (控制电源)

电池 7.4V- ──→ 公共 GND (所有模块的 GND 都连到这里)
```

- 如果是双联开关 (两个开关一体)，公共端直接并联
- 如果是两个独立开关，电池正极用一分二线分别接到两个开关

### 1.3 检查

- [ ] 万用表测开关1 输出端：开=7.4V，关=0V
- [ ] 万用表测开关2 输出端：开=7.4V，关=0V
- [ ] 万用表测 MP1584 输出端：5.0V ±0.1V
- [ ] 断开所有开关

---

## 第2步：TB6612 电机驱动板接线

**目标**: 连接电源和电机，测试电机转动。

> WHEELTEC R3X 套件的 TB6612 驱动板可能已预接好电机线，检查一下。

### 2.1 TB6612 电源接线

```
开关1 ON ──→ TB6612 VM+ (电机电源 7.4V)
公共 GND  ──→ TB6612 GND
```

### 2.2 TB6612 信号线接 LaunchPad

```
LaunchPad    →    TB6612
─────────────────────────
PB0          →    AIN1
PB1          →    AIN2
PB2          →    BIN1
PB3          →    BIN2
PB4          →    STBY
PA12         →    PWMA
PA13         →    PWMB
```

**接线方法**: 杜邦线一端插 LaunchPad 排针，另一端插 TB6612 对应引脚。

### 2.3 电机接线

```
TB6612 AO1  →  左电机+  (红线)
TB6612 AO2  →  左电机-  (黑线)
TB6612 BO1  →  右电机+  (红线)
TB6612 BO2  →  右电机-  (黑线)
```

> 如果电机转向不对，交换同一通道的两根线即可 (AO1↔AO2)。

### 2.4 不装轮胎测试

- 开关1 打开 (电机电源)
- 万用表测 TB6612 VM 端：应为 7.4V
- 暂不下载程序，手动将 PB0 接 3.3V (AIN1=H)，PB1 接 GND (AIN2=L)
- 电机应转动 (如不转，检查 STBY 是否接 HIGH)
- 断电

---

## 第3步：编码器接线

**目标**: 连接编码器信号线。

### 3.1 找到编码器线

WHEELTEC R3X 底盘的编码器线通常已从电机引出，一般是 6 线：
- 电机+、电机- (已在第2步接好)
- 编码器 VCC (红)、GND (黑)、A相 (绿/黄)、B相 (白/蓝)

> 颜色因厂家不同可能不同，看线上的标注或卖家说明。

### 3.2 编码器接线

```
编码器 VCC  →  LaunchPad 3.3V
编码器 GND  →  LaunchPad GND
左编码 A相  →  PC0
左编码 B相  →  PC1
右编码 A相  →  PC2
右编码 B相  →  PC3
```

> 两个编码器的 VCC/GND 都接到 LaunchPad 的 3.3V/GND。

---

## 第4步：巡线传感器接线

**目标**: 安装 7 个 TCRT5000 模块到车头。

### 4.1 物理安装

7 个 XD-01 模块并排安装在车头底部，间距约 1.0-1.5cm，传感器离地 3-5mm。

```
车头底部 (俯视):
┌──────────────────────────────┐
│  [S1] [S2] [S3] [S4] [S5] [S6] [S7]  │
│   PA2  PA5  PA6  PA7  PA8  PB9  PB10 │
└──────────────────────────────┘
```

### 4.2 每个 XD-01 模块接线 (×7)

```
XD-01 VCC  →  LaunchPad 3.3V
XD-01 GND  →  LaunchPad GND
XD-01 DO   →  对应 GPIO 引脚 (见下表)
XD-01 AO   →  不接 (悬空)
```

| 模块 | VCC | GND | DO 接到 |
|------|-----|-----|---------|
| S1 (最左) | 3.3V | GND | **PA2** |
| S2 | 3.3V | GND | **PA5** |
| S3 | 3.3V | GND | **PA6** |
| S4 (中) | 3.3V | GND | **PA7** |
| S5 | 3.3V | GND | **PA8** |
| S6 | 3.3V | GND | **PB9** |
| S7 (最右) | 3.3V | GND | **PB10** |

### 4.3 阈值调节

- 上电后，将传感器对准白纸，调节模块上蓝色电位器
- 使 DO 输出在白纸上为 LOW，黑线上为 HIGH
- 7 个模块逐一调节

---

## 第5步：舵机云台接线

**目标**: 组装二维云台并连接 MG90S。

### 5.1 云台组装

- 将两个 MG90S 舵机安装到自制云台支架上
- 一个控制水平旋转 (X轴)，一个控制俯仰 (Y轴)
- 激光模块安装在云台末端

### 5.2 舵机接线

```
MG90S 棕线 (GND)  →  公共 GND
MG90S 红线 (VCC)  →  MP1584 5V 输出
MG90S 橙线 (信号)  →  LaunchPad 引脚
```

| 舵机 | 棕线(GND) | 红线(VCC) | 橙线(信号) |
|------|-----------|-----------|-----------|
| 水平(X) | GND | MP1584 5V | **PA3** |
| 俯仰(Y) | GND | MP1584 5V | **PA4** |

> **重要**: 舵机 VCC 接 MP1584 5V，不接 LaunchPad 5V (电流不足)。
> 舵机 GND 必须与 LaunchPad GND 连通 (共地)。

---

## 第6步：激光模块接线

**目标**: 连接激光模块到云台。

### 6.1 激光模块接线

```
激光 VCC   →  LaunchPad 3.3V (如模块为 3.3V 规格)
             或 MP1584 5V (如模块为 5V 规格)
激光 GND   →  LaunchPad GND
激光 信号  →  100Ω 电阻 → PA20
```

> 先确认激光模块规格再选供电。大部分 405nm 模块为 3.3V。
> 如模块自带限流电阻，可省略外接 100Ω。

---

## 第7步：按键与 LED 接线

**目标**: 接入模式选择和启动按钮。

### 7.1 按键接线

```
按键一端  →  LaunchPad 引脚
按键另一端 →  GND
```

| 按键 | 引脚 | 说明 |
|------|------|------|
| MODE | **PB8** | 一端接 PB8，另一端接 GND |
| START | **PB11** | 一端接 PB11，另一端接 GND |

> 代码中配置内部上拉，按下=LOW，松开=HIGH。

### 7.2 LED 接线

```
LED 长脚(+) → 220Ω 电阻 → PA15
LED 短脚(-) → GND
```

---

## 第8步：OpenMV 接线 (最后接)

**目标**: 连接 OpenMV 通信线。

### 8.1 OpenMV 供电

```
OpenMV VIN  →  MP1584 5V 输出
OpenMV GND  →  公共 GND
```

### 8.2 UART 接线

```
OpenMV P5 (TX) → PA11 (MSPM0 RX)
OpenMV P4 (RX) → PA10 (MSPM0 TX)
```

> **注意**: 拔掉 LaunchPad 上 J21 和 J22 跳线帽，断开调试器与 UART0 的连接。

### 8.3 OpenMV 摄像头朝向

- OpenMV 摄像头朝向靶面方向
- 安装在云台上方或车体前端
- 确保镜头不被舵机/激光遮挡

---

## 全部接线完成 — 最终检查清单

### 电源检查 (万用表)

- [ ] TB6612 VM: 7.4V
- [ ] MP1584 输出: 5.0V
- [ ] LaunchPad 3.3V 引脚: 3.3V
- [ ] 各模块 GND 互通 (万用表通断档)

### 信号线检查 (对照接口图)

- [ ] PB0→AIN1, PB1→AIN2, PB2→BIN1, PB3→BIN2
- [ ] PB4→STBY, PA12→PWMA, PA13→PWMB
- [ ] PA3→舵机X, PA4→舵机Y
- [ ] PA2,PA5-PA8,PB9,PB10 → 巡线 S1-S7
- [ ] PC0-PC3 → 编码器
- [ ] PA10→OpenMV RX, PA11→OpenMV TX
- [ ] PA20 → 激光
- [ ] PB8→MODE, PB11→START, PA15→LED

### 无短路检查

- [ ] 3.3V 与 GND 之间无短路
- [ ] 5V 与 GND 之间无短路
- [ ] 7.4V 与 GND 之间无短路

## 接线顺序总结

```
第1步  电源模块 (MP1584 + 双路开关)     ← 先搭电源
第2步  TB6612 + 电机                     ← 验证电机能转
第3步  编码器                            ← 验证脉冲计数
第4步  巡线传感器 ×7                     ← 验证黑白检测
第5步  舵机 ×2                           ← 验证 PWM 控制
第6步  激光模块                          ← 验证开关控制
第7步  按键 + LED                        ← 验证中断输入
第8步  OpenMV                            ← 最后接通信
```

> 每一步接好后，建议先测试再进入下一步。测试方法见 [硬件接口图.md](硬件接口图.md) 第六节。
